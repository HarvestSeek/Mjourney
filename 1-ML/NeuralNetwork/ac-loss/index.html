    2. functions

    $1.\ \text{activation function}$

    激活函数对于神经网络，其目的是为了引入非线性

    $\text{(1). sigmoid}$




       $\large\displaystyle\mathrm{sigmoid}(x)=\frac{1}{1+e^{-x}}$




    • 优点:
      • $f'(x)=f(x) \cdot (1-f(x))$
      • 可将输入缩放到 $(0,1)$ 内
    • 缺点
      • 左右两端导数趋于 $0$，由 $\small\displaystyle \frac{\partial C}{\partial w}=\frac{\partial C}{\partial y} \frac{\partial y}{\partial \beta} \frac{\partial \beta}{\partial w},\ \frac{\partial y}{\partial \beta}\to 0$，易造成梯度消失 (gradient vanishing)，函数两端激活性差
      • $\text{non-zero-centered}$，i.e. 不关于原点中心对称。以如图所示的简单网络为例，梯度为
            $\small\displaystyle \frac{\partial C}{\partial w_i}=\frac{\partial C}{\partial y} \frac{\partial y}{\partial \beta} \frac{\partial \beta}{\partial w_i}=\frac{\partial C}{\partial y} f'(\beta)\ x_i$
        对于 $w_1,\ w_2$，前两项相当于常数，因此 $x_i$ 的正负决定 $w_i$ 的梯度的正负。如果 $\bm{x}$ 是通过 $\text{sigmoid}$ 输出的隐层，那么 $x_i$ 恒大于 $0$，$w_i$ 的更新都是同步的，这样就可能发生 $(w_1,w_2)$ 以 "Z" 字形趋近 $(w_1^*,w_2^*)$ 的情况

    $\text{(2). tanh}$




       $\large\displaystyle\mathrm{tanh}(x)=\frac{e^x-e^{-x}}{e^x+e^{-x}}$




    双曲正切函数将实值压缩到 $(-1,1)$ 之间，解决了 $\text{sigmoid}$ 零中心的问题，但仍存在两端梯度饱和的问题，且函数的计算更为复杂。

    $\text{(3). ReLU}$




       $\large\displaystyle\mathrm{ReLU}(x)=\mathrm{max}(0,x)$




    • 优点:
      • 计算简单，速度快
      • 大于 $0$ 处导数为常数，没有梯度弥散现象
      • 负半区导数为 $0$，一旦神经元激活值为负，即意味着它不会经历训练，引入稀疏性
    • 缺点
      • dead ReLU: ReLU 在训练过程中很脆弱。假设网络中初始化的 $w_{ij}$ 大部分 $0$，但学习率设置的过大，某次反向传播可能就导致一部分 $w_{ij}$ 小于 $0$；下一次正向传播就会导致一部分神经元 "死亡"，参数永远不会更新。

    $2.\ \text{loss function}$

    $\text{(1). Mean Squared Error Loss}$

    均方误差 (MSE) 又称 L2 loss，其基本形式为:

    $\displaystyle J_{\mathrm{MSE}} = \frac{1}{N} \sum_{i=1}^N (y_i - \hat{y_i})^2 $

    假设误差服从正态分布 $\Delta y \sim N(0, \sigma^2)$，即 $y_i\sim N(\bm{w^\mathrm{T}}\bm{x_i},\sigma^2)$，则给定 $\bm{x_i}$ 输出 $y_i$ 的概率为:

    $\displaystyle P(y_i|\bm{x_i}\ ;\bm{w}) = \frac{1}{\sigma \sqrt{2\pi}} \mathrm{exp} \left( - \frac{(y_i-\hat{y_i}) ^ 2} {2\sigma^2} \right) $

    进一步假设 $N$ 个样本独立，则其对数似然函数为

    $\begin{aligned} \displaystyle L(\bm{w}) &= \ln \prod_{i=1} ^ N \frac{1}{\sigma \sqrt{2\pi}} \mathrm{exp} \left( - \frac{(y_i-\hat{y_i}) ^ 2} {2\sigma^2} \right)\\ &= - \frac{1}{2 \sigma^2} \sum_{i=1}^N (y_i - \bm{w^\mathrm{T}}\bm{x_i}) ^ 2 - N\ln \sigma \sqrt{2\pi} \\ &\Rightarrow \sum_{i=1}^N (y_i - \hat{y_i})^2 \end{aligned} $

    因此，在误差服从正态分布的假设下，最小化均方误差和 MLE 本质一样。回归问题一般近似满足此条件，因此常使用 MSE，而分类问题则不使用

    $\text{(2). Mean Absolute Error Loss}$

    平均绝对误差 (MAE) 又称 L1 loss，其基本形式为:

    $\displaystyle J_{\mathrm{MAE}} = \frac{1}{N} \sum_{i=1}^N |y_i - \hat{y_i}| $

    类似地，假设误差服从拉普拉斯分布 ($\displaystyle\mu = 0, b = 1,\enspace \ f(x)= \frac{1}{2b} \mathrm{exp} \left( - \frac{|x - \mu|} {b} \right))$，最小化绝对误差和 MLE 本质一样

    $\text{(3). Cross Entropy Loss}$

    考虑多分类问题，$\bm{y_i}$ 是一个 one-hot 向量，输入经过 softmax 处理后得 $\bm{\hat{y_i}}=\lbrace \hat{y}^k_i\rbrace_1^K$，整个网络可以看作 给定 $x_i$ 的类别的概率分布

    $\displaystyle P(y=y_i|x_i\ ;\theta) = \prod_{k=1}^K (\hat{y}^k_i) ^ {y_i^k} $

    其对数似然为 ($c_i$ 为 $x_i$ 的目标类)

    $\displaystyle P(y_i|x_i;\ \theta) = \sum_{i=1}^N \sum_{k=1}^K {y_i^k} \log (\hat{y}^k_i) = \sum_{i=1}^N {y_i^{c_i}} \log (\hat{y}^{\hat{c}_i}_i) = - J_{\mathrm{CE}} $

    即最小化交叉熵等价于 MLE。
    交叉熵损失也可以从 KL 散度方面理解。考虑最优分布 $\bm{y}_i^ * $ 和输出分布 $\hat{y_i}$ 之间的 KL 散度:

    $\displaystyle \mathrm{KL}(\bm{y}_i^ * , \hat{y_i}) = \sum_{i=1}^K y_i^ * \log (y_i ^ * ) - \sum_{i=1}^K y_i^ * \log (\hat{y_i} ) $

    上式第一项是最优分布的信息熵，第二项是二者的交叉熵。第一项可看作常数，最小化 KL 散度即最小化第二项；再把 $\bm{y_i}$ 看作最优分布，上式即转化为了最小化交叉熵 loss。
